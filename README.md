# 🎵 Antify - Reproductor de Música

**Antify** es un reproductor de música web sencillo e intuitivo desarrollado con HTML, CSS y JavaScript.

---

## 🌟 Características

* 🎧 Reproducción de audio en tiempo real.
* 🖼️ Carátula e interfaz visual del reproductor.
* 💻 Compatible con navegadores web modernos.

---

## 🛠️ Tecnologías utilizadas

* **HTML5** (Estructura y reproducción de audio)
* **CSS3** (Diseño y estilos)
* **JavaScript** (Lógica de control)

---

## 🚀 Cómo probarlo localmente

1. Clona o descarga este repositorio.
2. Abre el archivo `index.html` en tu navegador favorito.
